# 生鲜商城系统简介 python63

## 项目概述


## 技术栈

- 开发语言：Python 3.11
- 后端框架：Django 5.0.6
- 数据库：MySQL 8.0
- 前端框架：Vue2

## 功能模块

### 客户端

- **登录/注册**：集成了短信服务功能。
- **首页**：展示热销商品和推荐商品。
- **搜索功能**：用户可以输入关键词搜索商品。
- **商品详情**：展示商品的图片、详细描述等信息。
- **订单结算**：通过购物车进行订单的生成和结算。
- **订单中心**：查看订单状态、订单详情及订单历史。
- **收货地址管理**：用户可以添加、修改或删除收货地址。
- **用户收藏**：用户可以将商品添加至收藏夹，也可管理收藏夹中的商品。
- **用户反馈**：用户可提出投诉和建议。

### 管理端

- 系统使用Django自带的admin管理后台，并采用simpleUI库进行界面美化，具备了全面的数据管理功能。

## 系统角色

- 用户：普通用户和管理员
- 管理员：负责商品的维护、订单管理以及用户反馈的处理。

## 运行环境

- Python 3.11+
- Django 5.0.6
- MySQL 8.0
- Vue2

## 部署步骤

部署步骤详细说明如下：

1. 环境准备：确保Python、MySQL等依赖环境正确安装。
2. 数据库迁移：执行迁移脚本来创建和更新数据库表结构。
3. 后端部署：配置好服务器和域名后，部署后端代码。
4. 前端部署：将前端代码部署至相应的Web服务器。
